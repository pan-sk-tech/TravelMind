## 判断

- `total_hotels=800`、`total_attractions=200`、`total_meals=520` 里，只有能被所选工具项/输出项加总解释的才算自然命中。
- `total_transportation=180` 没有路线工具价格基础，且在 few-shot 组明显高频，仍然更像提示词数字锚定。
- 如果同一条样本同时命中多个示例数字，尤其包含 `total_transportation=180`，需要优先人工审查。
