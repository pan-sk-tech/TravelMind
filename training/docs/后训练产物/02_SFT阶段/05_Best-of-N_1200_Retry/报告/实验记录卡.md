# 实验记录卡：Best-of-N 1200 Retry

## 一、阶段目标

Best-of-N 1200 Retry 的目标是扩大 best-of-n 数据规模，并重新比较 checkpoint，判断 1200 数据是否能稳定超过 600 主线，以及最终 checkpoint 是否一定最好。

## 四、阶段结论

1. `ckpt104` 的 softpass 和重算预算 softpass 最好，但 hardpass 低于 old600final。
2. `final1200` 不如 `ckpt104`，说明最终 checkpoint 不一定最好。
3. 1200 数据没有全面压过 600 final，不同 checkpoint 各有强项。
4. 继续 SFT 的边际收益开始变钝，后续更应该转向 rerank、预算重算和高质量偏好数据。

## 五、后续影响

Best-of-N 1200 是 SFT 收束前的重要证据：扩大 SFT 数据能微调子指标，但很难继续带来全局跃迁。后续默认不再常规追加 SFT 轮次，转向 rerank-first 和 DPO/RL。
