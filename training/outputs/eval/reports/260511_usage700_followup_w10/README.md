# 260511 Usage700 Follow-up Eval Report

This report package summarizes the two latest usage700 SFT follow-up models against the previous mainline and historical references.

Files:

- `中文结果摘要.md`: human-facing Chinese decision report.
- `replay_usage700_vs_legacy_b_baseline_full_report.md`: full report for the current best SFT candidate, `replay_usage700_lr2e5`, against legacy_b and baseline.

Source comparison artifacts were generated under `training/outputs/eval/comparisons/`, but only the lightweight human-facing reports above are included in this package.

Large artifacts such as `generations.jsonl`, full `rule_eval_report.json`, and service logs are intentionally not copied into this report package.
