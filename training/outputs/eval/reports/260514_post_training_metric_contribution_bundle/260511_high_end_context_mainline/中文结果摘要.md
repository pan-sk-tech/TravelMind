# 高端 POI 上下文重构后三模型中文结果摘要

本轮评测使用重构后的 standard / hard 评估集上下文，并已修正 POI 名称归一化口径。模型角色按这里理解：

- `base`：原始基线。
- `legacy_b`：旧版本数据路线上的历史最好版本，只作为历史强对照，不作为当前主线。
- `valloss_lr8e5`：当前新上下文 / 新数据路线候选，后文简称 `lr8e-5`。

## 总体判断

当前主线继续看 `lr8e-5`。它在 standard 的 hardpass 和 softpass 都最高；hard 集 hardpass 略低于历史强对照 `legacy_b`，但 softpass 与预算相关指标整体更好。`legacy_b` 固定为旧数据路线的历史最好对照，不作为新主线。

## 简短结论

- hardpass：`lr8e-5` 在 standard 最强；hard 集 `legacy_b` 最高，但它是旧数据历史强对照，`lr8e-5` 相比 base 仍大幅提升。
- softpass：`lr8e-5` 在 standard / hard 都高于 `legacy_b`，说明新路线的质量项收益更好。
- 预算：`lr8e-5` 在预算偏好、预算结构关系、餐饮价格尺度、hard 集重算预算贴合上整体更好；hard 集纯“不超预算”不是最高，但 base 的高分主要来自保守选择。
